## Synopsis

`rcptto_timeout = 300`

## Description

The amount of time in seconds to wait for a response to the `RCPT TO` command. The default value is `300`.

<a name="idp25911440"></a>
## Scope

rcptto_timeout is valid in the binding, binding_group, domain, and global scopes.
