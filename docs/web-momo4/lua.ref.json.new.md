## Name

json.new — Create an empty JSON object

<a name="idp16553392"></a>
## Synopsis

`require("json");`

`json.new()`

<a name="idp16556352"></a>
## Description

Create an empty JSON object.

<a name="lua.ref.new.example"></a>

**Example 70.38. json.new example**

```
o = json.new();
o.name = "value";
print(o); -- { "name": "value" }
```
