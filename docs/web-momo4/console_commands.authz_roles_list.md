## Synopsis

`authz roles list`

<a name="idp14201424"></a>
## Description

This command displays all the defined roles in XML format.

Execute this command and you should see output similar to the following:

```
<Roles>
 <Role name="users"/>
 <Role name="admin"/>
 <Role name="ecuser"/>
 <Role name="fred"/>
</Roles>
```
