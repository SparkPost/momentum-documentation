<a name="console_commands.pid"></a>
## Name

pid — show process id of Momentum

## Synopsis

`pid`

<a name="idp16216288"></a>
## Description

This command shows the process id of the running Momentum instance.

```
10:47:35 /tmp/2025> pid
481
```
