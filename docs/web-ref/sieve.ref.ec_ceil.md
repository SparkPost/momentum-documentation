## Name

ec_ceil — round up to the nearest integer

## Synopsis

`ec_ceil` { *`number`* }

<a name="idp29239216"></a>
## Description

`ec_ceil` rounds the given number up to the nearest integer.

<a name="example.ec_ceiling"></a>

**Example 16.29. ec_ceil example**

```
$a = 65.43;
$b = -41.65;
$c = 43;
$ceila = ec_ceil $a;
$ceilb = ec_ceil $b;
$ceilc = ec_ceil $c;
#ceila is 66.000000
#ceilb is -41.000000
#ceilc is 43.000000
```
